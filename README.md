# Hotel Management System

A Python-based desktop application for managing hotel operations with Tkinter GUI. Provides complete guest check-in/check-out functionality, room management, and data persistence.

## Features

### Guest Management
- Complete check-in/check-out system
- Guest details tracking (name, phone number)
- Automatic timestamp recording for all transactions
- Search functionality by name or phone number

### Room Management
- Room type classification (Single, Double, Deluxe, Suite)
- Unique room number assignment
- Real-time occupancy status tracking
- Prevents duplicate room assignments

### Data Handling
- Automatic JSON data storage
- Data persistence between sessions
- Backup and restore functionality
